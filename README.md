## Overview

Personal developer portfolio showcasing **8+ years** of fullstack experience across healthcare, fintech, logistics, and public transit. Single-page layout with animated sections, service cards, and professional branding.

---

## Features

- **Animated hero** — Profile, title, CTA with Framer Motion
- **About section** — Professional summary
- **Services grid** — 8 capability cards (Frontend, Backend, DevOps, Auth, Testing...)
- **Smooth scroll navigation** — Section anchors
- **Dark theme** — Cyan accent on charcoal background
- **Responsive** — Mobile-first layout

---

## Tech Stack

React 18 · TypeScript · Vite · Tailwind CSS · Framer Motion · Heroicons

---

## Quick Start

```bash
git clone https://github.com/jonathasribeiro/personal-portfolio.git
cd personal-portfolio
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

### Production

```bash
npm run build
npm run preview
```

Deploy to Vercel, Netlify, or any static host.

## Sections

| Section | Content |

| Home | Hero, photo, title, contact CTA |

---

## Customization

- Update photo URL in `src/App.tsx`
- Edit copy in hero/about sections
- Add `public/resume.pdf` for download button
- Tweak colors in Tailwind config (`#4fc3f7` accent)
